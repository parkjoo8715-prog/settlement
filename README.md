# 정산시스템 솔루션 프로토타입

## 개요
- 정적 HTML/CSS/JS로 만든 정산 서비스 프론트 프로토타입
- 차수 선택 중복 허용
- 차수별 참석자/신입 여부 개별 설정
- 신입 3만원 선입금은 최종 정산에서 1회만 적용
- 브라우저 `localStorage` 자동 저장 지원

## 파일 구성
- `index.html`: 화면 구조
- `styles.css`: UI 스타일
- `app.js`: 정산 계산 로직 및 상태 관리

## 실행 방법
1. `index.html`을 브라우저에서 직접 연다.
2. 또는 정적 서버로 실행한다.

## 배포 방법
- GitHub Pages
- Netlify
- Vercel Static
- 사내 웹서버 정적 호스팅

## 현재 서비스화 반영 항목
- 계좌 유형/기관 선택 + 계좌번호 분리 입력
- 차수 선택형 정산 카드 구조
- 차수 중복 선택 허용
- 모바일 테이블 대응
- 접근성 라이브 영역 보강
- 복사 기능 폴백 지원
- 자동 저장 지원
